# IB-Alpha Research Skill v2.0

A GitHub-ready research skill/package for A-share and HK/US listed company deep-dive analysis using an investment-banking style workflow.

## What this package does
This skill standardizes equity research into:
1. Industry structure
2. Business model
3. Competitive moat
4. Financial quality
5. Valuation
6. Catalysts
7. Risks
8. Capital markets lens
9. Management & governance
10. Investment conclusion

## Repository structure
- `README.md` — overview and usage
- `SKILL.md` — the full reusable skill instructions
- `PROMPT_TEMPLATE.md` — copy-paste template for new company research
- `OUTPUT_TEMPLATE.md` — standardized output format
- `CHECKLIST.md` — diligence checklist
- `EXAMPLE_RUN_RZKJ.md` — example for 润泽科技
